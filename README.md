# My Adventure Webgame
## description
A text-and-choice-based webgame where you have to take right choices to survive and win in the least number of turns.
## How to play?
Simply click on the demo link and click start game to start a new game, then you will face multiple situations where you have to choose one of two options to survive.
## Important note
The idea of the story itself of the game was made before from 3 years ago as a python text base game so I had an idea to convert this gmae idea into a webgame.
So the idea is old but the project itself as a webpage was made from beggining to the end recently with my own hands only.
